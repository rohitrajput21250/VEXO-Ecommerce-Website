# VEXO-Ecommerce-Website
🛍️ VEXO — A futuristic streetwear &amp; workout gear e-commerce website built with HTML, CSS &amp; Vanilla JavaScript.

A fully interactive, modern e-commerce concept website for streetwear and workout gear.

## ✨ Features
- 🧭 Fully functional navigation with dropdown menus
- 🛒 Working cart with add/remove & quantity control
- 🔐 Sign In / Sign Up modal with form validation
- 🖼️ Product detail panel with size & color selection
- ❤️ Wishlist functionality
- 🔍 Category filtering (Men, Women, Trending, Seasonal, Accessories)
- 📱 About & Contact pages
- 🖱️ Custom cursor with smooth trailing ring
- 🎞️ Scroll reveal animations
- 📸 Real Unsplash product & hero photography
- 🌀 Marquee ticker strip

## 🛠️ Built With
- HTML5
- CSS3 (no frameworks)
- Vanilla JavaScript (no libraries)
